фоновые напоминания

Push-готовность

Без маленького сервера приложение не может само просыпаться закрытым. В архиве есть пример push-server-example для будущих настоящих фоновых уведомлений.

iphone

Как поставить

  1. Опубликуй папку приложения на HTTPS: Netlify, GitHub Pages, Cloudflare Pages или свой домен.
  2. Открой ссылку на iPhone в Safari.
  3. Нажми кнопку «Поделиться» и выбери «На экран “Домой”».
  4. Назови приложение FocusFlow и нажми «Добавить».
  5. Открой уже с иконки на домашнем экране и нажми «Напоминания».
