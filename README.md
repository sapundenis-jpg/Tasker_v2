# FocusFlow PWA v0.7

Терминальный mobile-first веб-планировщик в строгой ретро-tech эстетике.

## Что нового в v0.7

- Добавлен переключатель тем:
  - `Dark` — строгая тёмная тема;
  - `Retro` — более зелёный терминальный экран;
  - `Minimal` — спокойная строгая версия без лишнего свечения.
- Добавлен мягкий CRT / scanline-эффект.
- Все вкладки, плашки, карточки статуса и инфо-блоки приведены к стилю старых зелёных экранчиков.
- Обновлены иконки приложения в пиксельном стиле.
- Выбранная тема сохраняется локально.
- Сохранена совместимость с данными v0.6/v0.5/v0.4/v0.3/v0.2/v0.1.

## Запуск

```bash
cd focusflow-pwa-v0.7
python -m http.server 8000
```

Открыть:

```text
http://localhost:8000
```

## Установка на iPhone

1. Опубликуйте проект на HTTPS: GitHub Pages, Netlify, Cloudflare Pages или свой домен.
2. Откройте ссылку в Safari.
3. Нажмите «Поделиться».
4. Выберите «На экран “Домой”».
5. Назовите приложение FocusFlow и нажмите «Добавить».

## Обновление на GitHub Pages

Замените файлы в репозитории на файлы из `focusflow-pwa-v0.7`, затем сделайте commit. Если на iPhone остаётся старая версия, откройте сайт в Safari и обновите страницу пару раз — PWA может держать старый кэш.
